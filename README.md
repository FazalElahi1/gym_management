# Gym Management System

## Overview
This project is a Gym Management System developed in Python using Pandas for managing gym member records and membership subscriptions. The application provides functionality for adding, removing, and analyzing member data through a menu-driven command-line interface.

The system demonstrates practical data handling, record management, and basic analytics using Python and Pandas.

---

## Features
- Add new gym members
- Remove existing members
- Display all member records
- Calculate average member age
- Track membership distribution
- Generate total revenue insights
- Multiple membership plans
- Menu-driven command-line interface

---

## Tech Stack
- Python
- Pandas
- Data Analysis Concepts
